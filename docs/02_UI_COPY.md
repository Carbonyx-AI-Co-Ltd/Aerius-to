# Aerius.to UI Copy

## Purpose

This document contains the approved UI copy for the `Aerius.to` frontend product shell.

The copy must position Aerius as a cognitive AI workspace, not as a generic chatbot SaaS.

The first product direction is Medical. The page should clearly direct users to the current Medical Demo while presenting the broader future workspace direction.

## Voice and Tone Rules

The copy should feel:

* serious
* calm
* precise
* technically confident
* premium
* medically aware
* not hype-heavy
* not salesy
* not generic SaaS

The copy should never overpromise.

The page should feel like a serious AI system for domains where mistakes matter.
